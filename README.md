## APIs

### The Movie Database API

The Movie Database (TMDB) API v3 provides comprehensive access to movie, TV show, person, and image data. Discover movies and series by genre, rating, and release date. Search across all content types. Retrieve cast and crew credits, external IDs, keywords, reviews, recommendations, similar titles, trailers, and watch providers. Manage user accounts with ratings, watchlists, and favorites. Supports multiple languages and regions. 148 endpoints covering movies, TV series, seasons, episodes, people, collections, companies, networks, keywords, and more.

- **Human URL:** [https://developer.themoviedb.org](https://developer.themoviedb.org)
- **Base URL:** `https://api.themoviedb.org/3`
