# The Movie Database

The Movie Database (TMDB) API v3 provides comprehensive access to movie, TV show, person, and image data. 148 endpoints covering search, discovery, credits, recommendations, trending, watch providers, user ratings, watchlists, and more.

**Human URL:** [https://developer.themoviedb.org](https://developer.themoviedb.org)

**Base URL:** [https://api.themoviedb.org/3](https://api.themoviedb.org/3)
